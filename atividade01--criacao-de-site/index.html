Sobre a Organização Tirulipa

A Organização Tirulipa surgiu graças aos Estudos Blessed após estes terem promovido uma atividade ao qual era nescessário criar um site em HTML.

Esse site foi feito individualmente pelo estudante Ian dos Santos, cursando Desenvolvimento de Sistemas ( Informática ), para testar seus conhecimentos de HTML.
